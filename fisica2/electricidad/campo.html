Campo

Magnitud física que tiene un valor definido para cada punto del espacio en cada instante de tiempo.

Suponiendo que tenemos algún sistema con cargas, definimos el campo eléctrico en un punto como 'la fuerza que recibiría una carguita muy chiquita'.

\begin{align} \vec E(\vec r) = \lim_{+q_0 \to 0} {\vec F_{+q_0} (\vec r) \over +q_0 } \\ \vec E(\vec r) = k \sum_{i=1}^{N} \frac{q_i'}{|\vec r - \vec r'_i|^3} (\vec r - \vec r'_i)\\ \end{align}
  • \(\vec r\) Punto campo: donde quiero calcular el campo
  • \(\vec r'_i\) Punto fuente: donde está la carga que 'produce' campo.

Podemos utilizar el campo eléctrico para encontrar la fuerza que percibe una carga en un punto.

\begin{align} \vec F_q(\vec r) = q.\vec E(\vec r) \end{align}

Lineas de campo: Ayuda para visualizar el campo.

  • Cargas positivas como fuentes
  • Cargas negativas como sumideros
  • La intensidad del campo es proporcional a la densidad de las líneas de campo.
  • El vector campo eléctrico es tangente a las líneas de campo en cada punto
